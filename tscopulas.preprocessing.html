tscopulas.preprocessing.lag_transform.apply_lag_transform(data, lag: int, axis: int = 0, drop: bool = False, fill: Optional[Any] = nan)¶

N-dimensional transform function which returns an N+1-dimensional matrix such that the last dimension has axis length <lag + 1> and the i’th index of the last dimension is the input matrix lagged by i

\[Let data be X \in \mathbb{R}^n, f(X) = Y such that Y \in \mathbb{R}^{n+1} and Y[...,i] = shift(X, i), # noqa i \in \{0, ..., lag\} # noqa\]
